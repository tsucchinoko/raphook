# Raphook

Git hook manager for Node.js projects inspired by `lefthook`.

## Install

With **NPM**:

```bash
npm install raphook --save-dev
```

## Usage

#### TL;DR

```bash
# Configure your hooks
vim raphook.yml

# Install them to the git project
raphook install

# Enjoy your work with git
git add -A && git commit -m '...'
```
